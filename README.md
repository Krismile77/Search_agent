# 🚀 AI 性能对比分析平台 - 多模型集成

一个集AI新闻爬虫、性能对比分析、API接口调用、快捷入口Web界面于一体的综合平台。

## ✨ 功能特性

- **📰 AI新闻爬虫** - 自动跟踪国内外AI大厂的模型发布动态
- **📊 性能对比分析** - 8个维度的AI模型性能评分
- **🔌 API接口** - 标准化的多模型调用接口
- **🌐 Web界面** - 美观的可视化展示 & 快捷入口
- **💬 模型对话** - 支持直接与多个模型对话
